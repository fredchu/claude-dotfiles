# 美股財報會議專業術語庫 (Glossary)

本文件包含美股財報電話會議中常用的核心財務、營運及技術術語，翻譯時必須採行「中文翻譯 (English Original)」格式。

## 1. 財務基礎指標
- **Low single digits** → 低個位數 (low single digits，約1-5%)
- **Mid double digits** → 中雙位數 (mid double digits，約30-60%)
- **High single digits** → 高個位數 (high single digits，約6-9%)
- **Year-over-year (YoY)** → 與去年同期相比 (YoY)
- **Quarter-over-quarter (QoQ)** → 與上一季相比 (QoQ)
- **Earnings per share (EPS)** → 每股盈餘 (EPS)
- **Generally Accepted Accounting Principles (GAAP)** → 一般公認會計原則 (GAAP)
- **Non-GAAP** → 非一般公認會計原則 (non-GAAP)
- **Free cash flow** → 自由現金流 (free cash flow)
- **Guidance** → 業績指引 (guidance)
- **Headwinds** → 挑戰 (headwinds)
- **Tailwinds** → 有利因素 (tailwinds)
- **Margin expansion** → 利潤率提升 (margin expansion)
- **Adjusted net income** → 調整後淨利 (adjusted net income)
- **Cost of goods sold (COGS)** → 銷售成本 (COGS)
- **Operating expenses (OpEx)** → 營運費用 (OpEx)
- **Operating leverage** → 營運槓桿 (operating leverage)

## 2. 營運與成長指標
- **Organic growth** → 有機成長 (organic growth)
- **Same-store sales** → 同店銷售 (same-store sales)
- **Comparable sales** → 可比銷售 (comparable sales)
- **Annual Recurring Revenue (ARR)** → 年度經常性收入 (ARR)
- **Recurring revenue** → 經常性收入 (recurring revenue)
- **Deferred revenue** → 遞延收入 (deferred revenue)
- **Bookings** → 訂單量 (bookings)
- **Backlog** → 訂單積壓 (backlog)
- **Gross merchandise value (GMV)** → 商品交易總額 (GMV)
- **Customer acquisition cost (CAC)** → 客戶獲取成本 (CAC)
- **Lifetime value (LTV)** → 客戶終身價值 (LTV)
- **Retention rate** → 留存率 (retention rate)
- **Customer churn rate** → 客戶流失率 (churn rate)
- **Monthly active users (MAU)** → 月活躍用戶 (MAU)
- **Average revenue per user (ARPU)** → 每用戶平均收入 (ARPU)

## 3. 技術與行業特定
- **Software-as-a-service (SaaS)** → 軟體即服務 (SaaS)
- **Cloud revenue** → 雲端收入 (cloud revenue)
- **Wafer shipments** → 晶圓出貨量 (wafer shipments)
- **Foundry services** → 晶圓代工服務 (foundry services)
- **Node technology** → 製程技術 (node technology)
- **Design wins** → 設計訂單 (design wins)
- **Foot traffic** → 客流量 (foot traffic)
- **Inventory turnover** → 存貨週轉率 (inventory turnover)
- **Inventory shrink** → 存貨損耗 (inventory shrink)

## 4. 口語與隱喻 (法證翻譯規範)
- **Give me more color** → 請補充細節 (能否具體談談/能否提供更多背景/可否再拆解一下)
- **Net new** → 淨新增 (例：net new customers -> 淨新增客戶)
- **Bake in/into** → 納入標準流程 (事前預置在流程裡)
- **Baking period** → 驗證期 (或觀察期)
- **Pipeline** → 商機管線 (或銷售管線)
- **Soft landing** → 軟著陸
- **Double down** → 加倍投入
- **Net-net** → 總而言之
