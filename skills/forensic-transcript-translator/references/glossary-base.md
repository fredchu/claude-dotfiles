# 基底術語對照表 (Base Glossary)

本表為跨公司通用的財報電話會議術語。翻譯時每個公司還需動態擴充公司專屬術語。

## 財務指標
| English | 中文 |
|---------|------|
| Revenue | 營收 (revenue) |
| Earnings per share (EPS) | 每股盈餘 (EPS) |
| Free cash flow | 自由現金流 (free cash flow) |
| GAAP | 一般公認會計原則 (GAAP) |
| Non-GAAP | 非一般公認會計原則 (non-GAAP) |
| Adjusted EBITDA | 調整後 EBITDA (adjusted EBITDA) |
| Gross margin | 毛利率 (gross margin) |
| Operating margin | 營業利益率 (operating margin) |
| Net income | 淨利 (net income) |
| Adjusted net income | 調整後淨利 (adjusted net income) |
| Cost of goods sold (COGS) | 銷售成本 (COGS) |
| Operating expenses (OpEx) | 營運費用 (OpEx) |
| Capital expenditure (CapEx) | 資本支出 (CapEx) |
| Guidance | 業績指引 (guidance) |
| Outlook | 展望 (outlook) |

## 成長與比較
| English | 中文 |
|---------|------|
| Year-over-year (YoY) | 與去年同期相比 (YoY) |
| Quarter-over-quarter (QoQ) | 與上一季相比 (QoQ) |
| Low single digits | 低個位數 (約1-5%) |
| Mid single digits | 中個位數 (約4-6%) |
| High single digits | 高個位數 (約6-9%) |
| Low double digits | 低雙位數 (約10-15%) |
| Mid double digits | 中雙位數 (約30-60%) |
| Organic growth | 有機成長 (organic growth) |

| English | 中文 |
|---------|------|
| Annual Recurring Revenue (ARR) | 年度經常性收入 (ARR) |
| Gross Payment Volume (GPV) | 總支付金額 (GPV) |
| Gross merchandise value (GMV) | 商品交易總額 (GMV) |
| Monthly active users (MAU) | 月活躍用戶 (MAU) |
| Average revenue per user (ARPU) | 每用戶平均收入 (ARPU) |

| Churn rate | 客戶流失率 (churn rate) |
| Net revenue retention (NRR) | 淨收入留存率 (NRR) |
